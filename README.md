# Arghya Das — Developer Portfolio

A modern and responsive developer portfolio built to showcase my projects, skills, certifications, and engineering journey as a Computer Science student.

This portfolio focuses on clean UI/UX, smooth interactions, performance, and a premium user experience while highlighting my passion for software engineering and problem solving.

## ✨ Features

- Modern dark/light theme
- Fully responsive design
- Smooth animations and transitions
- Interactive project showcase
- Skills and certification sections
- Contact form integration
- Performance-focused frontend structure
- Clean and scalable code architecture

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Design
- UI/UX Principles

## 🚀 Goals of This Portfolio

- Showcase real-world projects
- Demonstrate frontend development skills
- Build a strong professional presence

## 📌 Future Improvements

- Backend APIs

- Performance optimization
